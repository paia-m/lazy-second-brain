# Start Here

This vault is a small, practical starting point for a Second Brain.

You do not need to understand every tool before you begin. The first session has one job: make the vault usable on your machine and capture a few real notes.

If you want to see how the starter docs connect, open [[VAULT_MAP]] in graph view.

## The First Session

1. Open this folder in Obsidian.
2. Open [[00 Inbox/first-run|First Run Prompt]].
3. Paste the prompt into your assistant.
4. Let the assistant tailor the setup to your machine and comfort level.
5. Capture three simple notes before changing the structure.

## Where Things Go

- Put unsorted thoughts in `00 Inbox`.
- Put active goals in `01 Projects`.
- Put ongoing responsibilities in `02 Areas`.
- Put references in `03 Resources`.
- Move finished or inactive material to `04 Archive`.

Do not reorganise early. Capture first, tidy later.
