# First Run Prompt

Paste this into Codex, Claude Code, or another assistant that can work with local files after cloning or downloading the public repo.

Public repo: https://github.com/paia-m/lazy-second-brain

Related docs: [[START_HERE]], [[SETUP]], [[GUARDRAILS]], and [[VAULT_MAP]].

```text
You are helping me set up this Lazy Second Brain vault.

Primary intent for this session: get the vault fully set up and usable on my machine. Do not drift into theory, tool comparisons, or general productivity advice unless it directly helps complete the setup.

First, tailor the setup to me:
1. Ask what operating system I am using.
2. Ask whether I already have this repo locally. If not, have me clone `https://github.com/paia-m/lazy-second-brain.git` or download the public repo.
3. Ask where this repo lives on my machine.
4. Ask my comfort level: beginner, comfortable with files, or comfortable with terminal commands.
5. Ask whether I already have Obsidian installed.

Then guide me through the setup. If you can create or edit files directly, do it. If you cannot access my machine, give me one clear manual step at a time, wait for me to confirm it is done, then continue.

Use the files already in this repo. Do not recreate the vault from scratch unless something is missing.

Check that these core components exist:
- 00 Inbox
- 01 Projects
- 02 Areas
- 03 Resources
- 04 Archive
- START_HERE.md
- SETUP.md
- GUARDRAILS.md
- 03 Resources/Tools
- VAULT_MAP.md

Then help me:
1. Open the vault in Obsidian.
2. Read START_HERE.md and GUARDRAILS.md.
3. Confirm how the folders are meant to be used.
4. Capture three simple notes in 00 Inbox.
5. Update SETUP.md with anything still manual or unfinished.
6. Open VAULT_MAP.md and confirm the starter graph links are visible.

When done, confirm the vault is usable and tell me exactly what to open next.
```

## First Captures

After setup, create three small notes in this folder:

- One question you want help learning.
- One link worth saving.
- One explanation the assistant gave that was useful.
