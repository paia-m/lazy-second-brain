# Guardrails

These rules keep the vault useful while it is still small.

Use these with [[START_HERE]], [[SETUP]], and [[00 Inbox/first-run|First Run Prompt]].

## Do

- Capture before organising.
- Keep notes plain and readable.
- Ask the assistant to explain decisions in beginner-friendly language.
- Keep manual steps explicit.
- Add tools only when they reduce friction.

## Do Not

- Do not build a complex taxonomy before you have notes.
- Do not install plugins before the basic vault works.
- Do not let the assistant drift into theory when setup is unfinished.
- Do not treat the first structure as permanent.

## Session Intent

The first assistant session has one primary intent:

> Get this vault fully set up and usable on the user's machine.

Everything else is secondary.
