# Setup Checklist

Use this checklist during the first assistant-guided setup session.

Related docs: [[START_HERE]], [[GUARDRAILS]], [[00 Inbox/first-run|First Run Prompt]], and [[VAULT_MAP]].

## Machine Setup

- [ ] Confirm operating system.
- [ ] Confirm where this vault lives on the machine.
- [ ] Confirm comfort level: beginner, comfortable with files, or comfortable with terminal commands.
- [ ] Install Obsidian if needed.
- [ ] Open this repo folder as an Obsidian vault.

## Vault Setup

- [ ] Read `START_HERE.md`.
- [ ] Read `GUARDRAILS.md`.
- [ ] Open `00 Inbox/first-run.md`.
- [ ] Run the first-run prompt with an assistant.
- [ ] Capture three real notes.
- [ ] Open [[VAULT_MAP]] and confirm the graph has starter links.

## First Test

The vault is working when:

- [ ] You can open it in Obsidian.
- [ ] You know where new notes go.
- [ ] You have at least three notes in `00 Inbox`.
- [ ] You can explain what `01 Projects`, `02 Areas`, `03 Resources`, and `04 Archive` are for.
- [ ] You know where to find [[03 Resources/Tools/Obsidian|Obsidian]], [[03 Resources/Tools/Graphify|Graphify]], and [[03 Resources/Tools/QMD|QMD]] notes.
