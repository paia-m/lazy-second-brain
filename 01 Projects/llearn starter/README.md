# llearn Starter Project

This project is the first active setup project for the vault.

Related docs: [[START_HERE]], [[SETUP]], [[GUARDRAILS]], and [[VAULT_MAP]].

## Goal

Get the vault usable on your machine with the least possible friction.

## Done Means

- Obsidian can open the vault.
- The folder structure makes sense.
- Three notes exist in `00 Inbox`.
- `SETUP.md` records anything still unfinished.
- [[VAULT_MAP]] shows links between the starter docs.

## Ignore Until Later

- Advanced plugins.
- Custom themes.
- Complex tagging systems.
- Perfect folder structure.

Useful references: [[03 Resources/PARA Method|PARA Method]], [[03 Resources/Tools/Obsidian|Obsidian]], [[03 Resources/Tools/Graphify|Graphify]], and [[03 Resources/Tools/QMD|QMD]].
