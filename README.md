# Lazy Second Brain

A beginner-friendly Obsidian vault starter for building a practical Second Brain with an LLM.

The goal is simple: clone this repo, open it in Obsidian, and let an assistant guide the first setup session. The vault starts with the core folders, starter notes, guardrails, and first-run checklist already in place.

## Quick Start

1. Clone this repo.
2. Install [Obsidian](https://obsidian.md) if you do not already have it.
3. Open this folder as an Obsidian vault.
4. Start with [[START_HERE]].
5. Open [[VAULT_MAP]] to see the seeded graph.
6. Paste the setup prompt from [[00 Inbox/first-run|First Run Prompt]] into your assistant.

## What Is Included

- `00 Inbox` for first captures and unsorted notes.
- `01 Projects` for active work.
- `02 Areas` for ongoing responsibilities.
- `03 Resources` for references and tool notes.
- `04 Archive` for inactive or complete material.
- Guardrails in [[GUARDRAILS]].
- First-run checklist in [[SETUP]].
- A graph-friendly map in [[VAULT_MAP]].

Keep the vault small at first. Capture what helps you learn, ignore what does not, and let structure emerge from use.
